\subsection{Descrição Detalhada}
An attribute is a name-\/value pair. Elements have an arbitrary number of attributes, each with a unique name. \begin{DoxyNote}{Observação}
The attributes are not X\+M\+L\+Nodes. You may only query the \doxyref{Next()}{pag.}{classtinyxml2_1_1_x_m_l_attribute_a7fd852d6185af90361ec1bc9a7681ad6} attribute in a list. 
\end{DoxyNote}
