\subsection{Descrição Detalhada}
Implements the interface to the \char`\"{}\+Visitor pattern\char`\"{} (see the Accept() method.) If you call the Accept() method, it requires being passed a \doxyref{X\+M\+L\+Visitor}{pag.}{classtinyxml2_1_1_x_m_l_visitor} class to handle callbacks. For nodes that contain other nodes (Document, \doxyref{Element}{pag.}{class_element}) you will get called with a Visit\+Enter/\+Visit\+Exit pair. Nodes that are always leafs are simply called with \doxyref{Visit()}{pag.}{classtinyxml2_1_1_x_m_l_visitor_adc75bd459fc7ba8223b50f0616767f9a}. If you return \textquotesingle{}true\textquotesingle{} from a Visit method, recursive parsing will continue. If you return false, {\bfseries no children of this node or its siblings} will be visited. All flavors of Visit methods have a default implementation that returns \textquotesingle{}true\textquotesingle{} (continue visiting). You need to only override methods that are interesting to you. Generally Accept() is called on the \doxyref{X\+M\+L\+Document}{pag.}{classtinyxml2_1_1_x_m_l_document}, although all nodes support visiting. You should never change the document from a callback. \begin{DoxySeeAlso}{Veja também}
